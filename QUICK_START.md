## Getting Started (5 minutes)

### Step 1: Download and Extract
- Download this folder to your computer
- Extract to a location you'll remember (e.g., `Documents/git-practice/`)

### Step 2: Install Tools (if needed)
1. **Git**: https://git-scm.com/downloads
   - Windows: Download and run installer
   - Mac: `brew install git` or download installer
   - Linux: `sudo apt-get install git`

2. **SourceTree**: https://www.sourcetreeapp.com/
   - Download and install (it's free!)
   - Skip Bitbucket sign-up if asked

### Step 3: Test the App
1. Open `index.html` in your web browser
2. Try adding tasks, completing them, deleting them
3. Toggle dark mode!

### Step 4: Start the Tutorial
Open `GIT_TUTORIAL.md` and follow along!
