# Task Manager App

A simple task manager web application for learning Git and SourceTree.

## Features
- Add new tasks
- Mark tasks as completed (click on them)
- Delete tasks
- Clean, modern UI

## How to Run
Simply open `index.html` in your web browser.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Learning Git with This Project
This project is designed for hands-on Git practice. You'll learn:
- Creating commits

## change from github
## another change from github
- Working with branches
- Merging changes
- Resolving conflicts
- Using SourceTree GUI

Happy Learning!
